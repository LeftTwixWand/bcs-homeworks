# Задача 1. Сумма цифр числа

## Описание

Напишите программу, которая запрашивает у пользователя целое число и выводит на экран сумму всех его цифр.

### Результат работы программы:

#### Пример 1

```
Введите целое число:
7345
Сумма цифр: 19
```

#### Пример 2

```
Введите целое число:
1234
Сумма цифр: 10
```
