# Задача 3. Подсчет слов

## Описание

Дан текстовый файл `input.txt`, в котором записан некоторый текст. Необходимо найти сколько раз встречается в этом тексте каждое слово и записать пары `слово - количество` в файл `output.txt`. Не забудьте обработать ошибки!

### Пример

```
Файл input.txt:
зеленая зелень зеленит зеленую зелень

Файл output.txt:
зеленая - 1
зелень - 2
зеленит - 1
зеленую - 1
```

### Подсказки

> Не читайте этот раздел сразу, попытайтесь сначала решить задачу самостоятельно :)

<details>

<summary>Подсказка 1</summary>

Очень удобно воспользоваться контейнером

```cs
Dictionary<string, int>
```

где ключ - слово, которое есть в тексте, а значение - сколько раз это слово встретилось.

</details>

<details>

<summary>Подсказка 2</summary>

Для того чтобы корректно вечти подсчет - надо проверять, встречалось это слово или нет.

Удобно сделать это вот таким кодом:

```cs
Dictionary<string, int> words = new Dictionary<string, int>();
// код...
if (words.ContainsKey(word)) // проверяем, есть ли ключ в контейнере
{
    words[word] += 1;
}
else
{
    words.Add(word, 1);
}
```
